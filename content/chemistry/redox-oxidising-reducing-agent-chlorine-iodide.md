## Question

Chlorine gas is bubbled through an aqueous solution of potassium iodide:

$$\text{Cl}_2(aq) + 2\text{KI}(aq) \rightarrow 2\text{KCl}(aq) + \text{I}_2(aq)$$

(a) State the oxidation state of chlorine and of iodine before and after the reaction.
(b) Identify the oxidising agent and the reducing agent, with a reason for each.

## Solution

**Step 1 — Oxidation state of chlorine.**

In \(\text{Cl}_2\), chlorine is an uncombined element, so its oxidation state is \(0\).

In \(\text{KCl}\), potassium (a Group I metal) is always \(+1\). Since the compound is neutral overall:

$$(+1) + (\text{Cl}) = 0 \quad\Rightarrow\quad \text{Cl} = -1$$

So chlorine goes from \(0\) in \(\text{Cl}_2\) to \(-1\) in \(\text{KCl}\) — a **decrease** in oxidation state.

**Step 2 — Oxidation state of iodine.**

In \(\text{KI}\), potassium is \(+1\), so iodine is \(-1\) (same reasoning as above).

In \(\text{I}_2\), iodine is an uncombined element, so its oxidation state is \(0\).

So iodine goes from \(-1\) in \(\text{KI}\) to \(0\) in \(\text{I}_2\) — an **increase** in oxidation state.

**Step 3 — Identify oxidation and reduction.**

A decrease in oxidation state (gain of electrons) is **reduction**; an increase (loss of electrons) is **oxidation**.

- Chlorine: \(0 \rightarrow -1\), oxidation state decreases → chlorine is **reduced**.
- Iodine: \(-1 \rightarrow 0\), oxidation state increases → iodine is **oxidised**.

**Step 4 — Name the agents.** The species that is *reduced* is the **oxidising agent** (it takes electrons from, and so oxidises, the other reactant). The species that is *oxidised* is the **reducing agent**.

- \(\text{Cl}_2\) is reduced → \(\text{Cl}_2\) is the **oxidising agent**.
- \(\text{KI}\) (the \(\text{I}^-\) ion) is oxidised → \(\text{KI}\) is the **reducing agent**.

**Check:** the total decrease in oxidation state must equal the total increase, since electrons are conserved. Chlorine: 2 atoms each drop by 1 unit = decrease of 2. Iodine: 2 atoms (from 2KI) each rise by 1 unit = increase of 2. The two balance, confirming the assignment is consistent with the balanced equation.

**Common mistake:** naming the oxidising agent as the species that is itself oxidised (i.e. confusing \(\text{Cl}_2\)/oxidising agent with "the thing that gets oxidised"). Remember: the oxidising agent causes oxidation *in the other reactant* by being reduced itself.

## Answer

(a) Chlorine: \(0 \rightarrow -1\) (reduced). Iodine: \(-1 \rightarrow 0\) (oxidised).
(b) Oxidising agent = **Cl₂** (it is reduced). Reducing agent = **KI** (it is oxidised).
